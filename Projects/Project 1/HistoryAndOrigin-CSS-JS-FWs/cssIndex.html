Historia y Orígenes

Primeramente las páginas web eran simple y sencillamente texto plano usando el Lenguaje de Marcado Generalizado Estándar, o mejor conocido como SGML. Inicialmente se sugieren dos propuestas. La primera se llamaba: Cascading HTML Style Sheets o CHSS y la segunda era Stream-based Style Sheet Proposal o SSP. Finalmente, los creadores Håkon Wium Lie y Bert Bos respectivamente, decidieron unir las mejores ideas de cada una de las dos propuestas y crear una sola llamada Cascading Style Sheets (CSS) como la conocemos hoy en día. En 1995, W3C decide implementar CSS al grupo de HTML e impartir una estandarización en 1996. Fue en ese momento cuando se creó la primera versión, llamada CSS Nivel 1. Posteriormente W3C decide dividir el grupo de trabajo de HTML en tres grupos diferentes: Grupo de Trabajo HTML, Grupo de Trabajo DOM (Modelo de Objetos del Documento) y Grupo de CSS. A partir de ahí, el grupo de trabajo de Cascading Style Sheets ha seguido publicando más versiones hasta la actualidad.

Versiones

Primeramente, en 1996 se publica la primera versión de CSS, mejor llamada como CSS Nivel 1. Posteriormente en 1998, la W3C (World Wide Web Consortium) publica CSS Nivel 2, con mejoras innovadoras. El CSS Nivel 2.1, después de varios años de tener su publicación rechazada, en el 2011 se publica exitosamente. Cascading Style Sheets por fin adquiere popularidad, y mucha más cuando en 1999, se lanza la versión nivel 3. Hasta hoy en día CSS Nivel 3 es el más avanzado, por ende el más usado. Sin embargo, el CSS Nivel 4 está en desarrollo hasta hoy en día.

Estándares

Actualmente existen dos estándares, CSS Nivel 1 y CSS Nivel 2. La primera especificacion del nivel 1 básicamente lo definen como obsoleto debido a que sus características y propiedades son las mismas del mismo nivel 1 de CSS; ya que este nivel tiene definido su sintaxis y definiciones en la especificación CSS 2.1. Mientras que CSS 2 era más estable, se decide que CSS 2.1 es mas recomendable. Sin embargo, algunas características de CSS 2 que fueron retiradas de CSS 2.1, fueron implementadas en CSS 3. El CSS Nivel 3 es construido módulo por módulo sobre CSS Nivel 2 utilizando la especificación CSS 2.1 cómo su núcleo.

¿Quién lo controla?

Primeramente creado por Håkon Wium Lie y Bert Bos con ayuda de World Wide Web Consortium, ahora mantenido y actualizado por uno de los tres grupos que se había dividido a partir del grupo de HTML llamado grupo de CSS.
