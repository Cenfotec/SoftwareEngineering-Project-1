Definición

El Pair Programming es un método de desarrollo ágil que se relaciona con el Extreme Programming (XP). El cual, se enfoca en una técnica en la que dos programadores trabajan juntos en una misma zona de trabajo y con una misma computadora, cada uno tiene un rol asignado durante el tiempo de trabajo, uno de ellos se le conoce como “Driver” que es el encargado de codificar y realizar el trabajo, y al otro se le conoce como “Navigator”, el cual es el que se encarga de revisar cada línea de código a medida que se vaya ingresando.
